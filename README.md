# 学生宿舍管理系统简介 java381

## 项目概述

学生宿舍管理系统是一款基于Springboot和Vue的前后端分离的信息管理系统。该系统采用MVC三层架构，以Spring、SpringBoot、Mybatis和Vue等技术开发，运用Maven作为项目构建工具，使用MySQL数据库进行数据存储。

## 技术栈

- **前端**: Vue
- **后端**: Springboot
- **数据库**: MySQL
- **项目构建**: Maven

## 功能模块

- 学生信息管理
- 宿舍楼信息管理
- 宿舍分配与调整
- 维修与投诉处理
- 数据统计与分析

## 系统角色

- 管理员
- 学生
- 宿管员

## 运行环境

- **操作系统**: Windows/Linux/Mac OS
- **服务器**: Tomcat 8.5+
- **数据库**: MySQL 5.7+
- **JDK**: JDK 1.8+

## 部署步骤

1. 搭建开发环境
   - 安装JDK 1.8或更高版本
   - 安装MySQL 5.7或更高版本
   - 安装Maven
   - 安装IDE（例如IntelliJ IDEA或Eclipse）
2. 克隆或导入项目代码
3. 编译和构建项目
4. 部署后端服务
5. 部署前端项目
